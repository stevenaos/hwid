Cara Install APK (Android)
Ikuti langkah berikut untuk install di Android
  1. 1
    Download file APK menggunakan tombol di atas
  2. 2
    Buka Pengaturan → Keamanan lalu aktifkan Install dari sumber tidak dikenal
  3. 3
    Buka file APK dari notifikasi atau folder Download
  4. 4
    Tap Install dan tunggu proses selesai
  5. 5
    Buka aplikasi, login dengan akun & key Anda
Cara Install IPA (iOS)
Pilih salah satu metode sideload berikut
  1. 1
    Download file IPA menggunakan tombol di atas
  2. 2
    Install AltStore atau Sideloadly di PC/Mac Anda
  3. 3
    Hubungkan iPhone/iPad ke PC via kabel USB
  4. 4
    Drag & drop file IPA ke AltStore / Sideloadly lalu klik Install
  5. 5
    Di iPhone: Pengaturan → VPN & Manajemen Perangkat → Trust developer
Persyaratan Android
  • Android 8.0 (Oreo) atau lebih baru
  • RAM minimal 2 GB
  • Ruang penyimpanan 50 MB
  • Koneksi internet aktif
  • Key akses aktif (beli di Key Shop)
Persyaratan iOS
  • iOS / iPadOS 14.0 atau lebih baru
  • iPhone 6s / iPad Air 2 atau lebih baru
  • Ruang penyimpanan 60 MB
  • AltStore / Sideloadly / TrollStore
  • Key akses aktif (beli di Key Shop)
